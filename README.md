### 📂 Избранные проекты

#### 🔥 [Foodgram — Продуктовый помощник](https://github.com/Chashkapluy1/foodgram-project-react)
*Fullstack-решение для публикации рецептов и создания списков покупок.*
*   **Задача:** Реализация сложной логики взаимодействия пользователей, подписок и ингредиентов.
*   **Ключевые фичи:**
    *   🛒 Формирование PDF-списка покупок на основе выбранных рецептов.
    *   👥 Система подписок на авторов.
    *   🔍 Фильтрация по тегам и ингредиентам.
*   **Стек:** `Django`, `DRF`, `PostgreSQL`, `Docker-compose`, `Nginx`.

#### 📚 [YaMDB — Платформа отзывов](https://github.com/Chashkapluy1/api_yamdb)
*REST API для сбора отзывов на произведения (фильмы, книги, музыка).*
*   **Особенность:** Командная разработка с жестким код-ревью.
*   **Ключевые фичи:**
    *   🔐 Кастомная система аутентификации (JWT) и ролей (User, Moderator, Admin).
    *   ⭐ Расчет рейтинга произведений.
*   **Стек:** `Django REST Framework`, `Simple JWT`, `PostgreSQL`.

#### ⚡ [Async YaCut — Сокращатель ссылок](https://github.com/Chashkapluy1/yacut)
*Высокопроизводительный асинхронный сервис.*
*   **Задача:** Обеспечить максимальную скорость обработки запросов.
*   **Ключевые фичи:**
    *   🚀 Полная асинхронность на FastAPI.
    *   🛡 Валидация URL и обработка коллизий.
*   **Стек:** `FastAPI`, `SQLAlchemy (Async)`, `Alembic`, `Uvicorn`.

#### 😺 [Kittygram — Соцсеть для питомцев](https://github.com/Chashkapluy1/kittygram)
*Учебный проект для отработки Docker-контейнеризации.*
*   **Задача:** Развертывание приложений в контейнерах.
*   **Стек:** `Django`, `Nginx`, `Gunicorn`, `Docker`.
